# react-native-draco

library for compressing and decompressing 3D geometric meshes and point clouds in React Native

## Installation

```sh
npm install react-native-draco
```

## Usage


```js
import { multiply } from 'react-native-draco';

// ...

const result = multiply(3, 7);
```
